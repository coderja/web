1.简介

 1.1 关于vuex官网讲述的更加详细，我这边简要的谈一下自己的理解，由于组件之间通信比较复杂，页面还显得比较繁琐，所以出现了vuex状态管理器，来集中管理组件之间共用的一些数据

2.使用场景

 2.1。如果你的项目不是大型的应用项目，使用vuex也是烦琐冗余的，反之则建议使用
